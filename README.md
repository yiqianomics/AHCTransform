# AHCTransform

Interactive GitHub Pages visualization for AHC and CLR transformations on zero-containing compositional data.

## GitHub Pages

This repository is ready to publish from the repository root:

1. Push the `main` branch to GitHub.
2. In GitHub, open **Settings** > **Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select branch `main` and folder `/ (root)`.
5. Save.

After GitHub finishes deploying, the site should be available at:

https://yiqianomics.github.io/AHCTransform/

## Local Preview

Because the app is plain HTML, you can open `index.html` directly in a browser, or serve the folder locally with any static file server.
